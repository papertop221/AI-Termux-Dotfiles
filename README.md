# 🌌 Termux AI Setup

Transformasikan terminal Termux Anda menjadi lingkungan kerja berbasis AI yang modern, bersih, dan minimalis.

## ✨ Fitur
- **Modern UI**: Menggunakan tema Catppuccin Macchiato.
- **Minimalist Prompt**: Didukung oleh Starship untuk tampilan yang ringkas dan informatif.
- **AI Integration**: Terintegrasi langsung dengan Gemini CLI.
- **Nerd Fonts**: Menggunakan JetBrainsMono Nerd Font untuk dukungan ikon yang lengkap.
- **Modern Tools**: Mengganti `ls` standar dengan `eza` yang berwarna dan berikon.

## 🚀 Cara Instalasi

1. Clone atau masuk ke direktori ini:
   ```bash
   cd ~/termux-ai-setup
   ```

2. Jalankan skrip instalasi:
   ```bash
   bash setup.sh
   ```

3. Setelah selesai, restart Termux atau jalankan:
   ```bash
   source ~/.bashrc
   ```

## ⌨️ Perintah Berguna
- `ai "<pertanyaan>"` : Berinteraksi dengan Gemini AI.
- `ls` : Melihat file dengan ikon modern.
- `ll` : Melihat detail file dengan tampilan list yang cantik.
